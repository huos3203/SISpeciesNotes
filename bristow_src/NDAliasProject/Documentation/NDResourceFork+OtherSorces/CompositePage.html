NDResourceFork+OtherSorces.h


Abstract

Defines the interface for a category of the class NDResourceFork.

Discussion

Defines method for NDResourceFork for retrieving resource type data from non resource fork sources, at lest not directly. The category OtherSorces can be omitted from your project if the additonal functionalty is not desired.
